# Task Manager

**Description**

Task Manager allows you to keep lists of your tasks. 
The task has a priority, end date, description, and hashtag. 
The user can add, modify, delete tasks,
search for tasks.

**Technologies**

- ASP.NET Core MVC
- HTML, CSS, JQuery, Apexcharts
- SQL Server
